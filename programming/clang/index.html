C のコードの実行結果を表示

 \   (ソースコードの表示)
つぎのように C のコードを貼り付けることができる.
#include <stdio.h>
int main(){
    printf("Hello Pika\n");
    return 0;
}
#include <stdio.h>
int main(){
    printf("Hello Pika\n");
    return 0;
}
#include <stdio.h>
int main(){
    printf("Hello Pika\n");
    return 0;
}
include の周りの < と > は取り扱いに注意が必要.

#include "stdio.h"
int main(){
    printf("Hello World\n");
    return 0;
}
