実対称行列

Definition 3.1.1 (実対称行列)
nnn 次正方行列 AAA, すなわち, A∈Matn(R)A\in\mathop{\mathrm{Mat}}_{n}(\mathbb R)A∈Matn​(R) に対して AT=AA^T = AAT=A を満たすものを対称行列と呼ぶ. 特に成分が実数であることを強調する場合は実対称行列と呼ぶ.

  • ここでは通常の線形代数学の講義で取り扱う対称行列が対角可能であることを認める.

Proposition 3.1.2 (実対称行列の対角可能性)
nnn 次対称行列 AAA は適当な直交行列により対角化できる. すなわち, 次の条件を満たす U∈O(n)U\in O(n)U∈O(n) 及び λ1,λ2,…,λn∈R\lambda_1,\lambda_2,\dots,\lambda_n\in\mathbb Rλ1​,λ2​,…,λn​∈R が存在する:

A=Udiag(λ1,λ2,…,λn)U⊤. A = U \mathop{\mathrm{diag}}(\lambda_1,\lambda_2,\dots,\lambda_n) U^\top. A=Udiag(λ1​,λ2​,…,λn​)U⊤.

さらに λi (1≤i≤n)\lambda_i\ (1\leq i \leq n)λi​ (1≤i≤n) は AAA の固有値になる.

二次形式

Definition 3.2.1 (二次形式)

  • AAA を nnn 次実対称行列とする. x∈Rn\bm{x}\in\mathbb R^nx∈Rn に対して qA(x)=x⊤Axq_A(\bm{x}) = \bm{x}^\top A \bm{x}qA​(x)=x⊤Ax を AAA が定める二次形式 (quadratic form) と呼ぶ. x=[x1,…,xn]⊤\bm{x}=\left[x_1,\dots,x_n\right]^\topx=[x1​,…,xn​]⊤, A=[aij]i,j=1nA=[a_{ij}]_{i,j=1}^{n}A=[aij​]i,j=1n​ と成分表示した場合に, qA(x)q_A(\bm{x})qA​(x) は次のようになる:

qA(x)=∑i,j=1naijxixj=∑i=1naiixi2+2∑1≤i<j≤naijxixj. q_A(\bm{x}) = \sum_{i,j=1}^n a_{ij}x_ix_j = \sum_{i=1}^n a_{ii}x_i^2 + 2\sum_{1\leq i < j\leq n} a_{ij}x_i x_j. qA​(x)=i,j=1∑n​aij​xi​xj​=i=1∑n​aii​xi2​+21≤i<j≤n∑​aij​xi​xj​.

半正定値/正定値行列

  • 半正定値/正定値行列行列の定義をおこなう.

Definition 3.3.1 (半正定値)
nnn 次実対称行列 AAA が半正定値行列であるとは, 任意の x∈Rn∖{0}\bm{x}\in \mathbb R^n\setminus \{\bm{0}\}x∈Rn∖{0} に対して qA(x)≥0q_A(\bm{x}) \geq 0qA​(x)≥0 が成り立つことである.

  • 要するに qAq_AqA​ の値が 0 以上であることを満たすことである.この条件を強くし, qAq_AqA​ の 0\bm{0}0 以外で真に 000 以上を満たす場合は正定値という(次の定義を参照).

Definition 3.3.2 (正定値)
nnn 次実対称行列 AAA が正定値行列であるとは, 任意の x∈Rn∖{0}\bm{x}\in \mathbb R^n\setminus \{\bf{0}\}x∈Rn∖{0} に対して qA(x)>0q_A(\bm{x}) > 0qA​(x)>0 が成り立つことである.

Proposition 3.3.3
nnn 次半正定値行列 AAA の固有値は全て 0 以上である.
